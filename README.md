> [!warning]

> **AIGC Warning**: This project is particially generated by using LLM

这是一个扩展 [TwitchDropsMiner](https://github.com/DevilXD/TwitchDropsMiner)，能够以更合适的方式启动以及使用 TDM 的一个扩展程序，在本项目中，TwichDropsMiner 作为 submodule 运行

如何理解这种模式呢，你可以看作 mihomo 核心与软件之间的关系，原来的项目 TDM 就是那个核心，而 TDMConsole 就是类似于 flclash 这样的一个便于使用的应用程序

关于掉宝相关的原理及其操作方式，请移步至原项目 [DevilXD/TwitchDropsMiner](https://github.com/DevilXD/TwitchDropsMiner)

## 开始使用

### 启动参数说明

| 参数                                 | 用途                                     | 必需  | 默认值                |
| ------------------------------------ | ---------------------------------------- | ----- | --------------------- |
| `-h`/`--help`                        | 展示帮助信息                             | False |                       |
| `-c`/`--config`                      | 指定配置文件                             | False | `./settings.json`     |
| `--game "game1,game2,..."`           | 指定游戏列表                             | False | `<EMPTY>`             |
| `--cookie COOKIE`                    | 将 Twitch 的 Cookie 在 TDMConsole 中复用 | False | `<EMPTY>`，会要求登录 |
| `--jar COOKIEJAR`                    | 指定 Cookie 文件的路径                   | False | `./cookies.jar`       |
| `--mode {web,repl,tui,gui,headless}` | 指定 TDMConsole 的运作模式               | False | `web`                 |
| `--host ADDRESS`                     | 设置 WEBUI 的监听地址                    | False | `127.0.0.1`           |
| `--port PORT`                        | 设置 WEBUI 的监听端口                    | False | `8080`                |
| `--v[vvv]`                           | verbose 调试输出                         | False |                       |
| `--log`                              | 写出日志到 `log.txt`                     | False |                       |
| `--version`                          | 展示版本号                               | False |                       |

### 二进制产物

直接通过 GitHub Release 页面下载最新的二进制产物，注意选择对应的系统和架构。

macOS 同时提供三种产物：`tdmconsole-macos-*` 是命令行程序，
`TDMConsole-macos-*.app.zip` 是 `.app` 压缩包，`TDMConsole-macos-*.dmg`
是可将应用拖入 Applications 的安装镜像。

程序分为 WebUI、REPL、TUI、GUI 四种可交互模式和纯日志的 headless 模式。
命令行程序在没有参数时默认启动 WebUI；macOS `.app` 在没有指定 `--mode`
时默认启动原生 GUI。`.app` 的运行数据保存在
`~/Library/Application Support/TDMConsole`，不会写入应用包内部。

在命令行调用即可启动程序（注：根据平台的不同，这里用到的程序名称也不同）

```bash
$ ./tdmconsole-macos-arm64
```

用法跟 TDM 没什么区别，如果你使用的 REPL 模式，可以通过 `/help` 命令查看可用的所有命令

### 源码运行

在装好 uv 的情况下，克隆本仓库并运行就好了

```bash
$ git clone --recursive https://github.com/MeowCracker/TDMConsole.git
$ cd TDMConsole
$ git submodule update --remote TwitchDropsMiner
$ uv sync
$ uv run main.py
```

其中，`git submodule update --remote TwitchDropsMiner` 用于跟原来的 TDM 进行同步，如果你不打算更新的话可以不用的
